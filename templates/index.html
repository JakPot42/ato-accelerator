📄 What is an ATO?

An Authority to Operate (ATO) is a formal declaration by a senior official that authorizes a federal information system to operate, accepting residual risk. Required before any DoD/federal system goes live.

📋 What is RMF?

The Risk Management Framework (NIST SP 800-37) is a 7-step process: Prepare → Categorize → Select → Implement → Assess → Authorize → Monitor. This tool covers steps 2–5.

📝 What is NIST 800-53?

The security and privacy control catalog for federal information systems. Rev 5 contains 1,000+ controls across 20 families. Your baseline (Low/Moderate/High) determines which controls apply.
